Panduan Modul

Cara pakai Admin Sistem

Panduan ini membantu Owner/SuperAdmin/Supervisor mengelola user, password hash, permission role, audit log, dan pengecekan master sheet ringkas.

1

Cek user sistem

Tabel user menampilkan username, nama, level, outlet, dan status password. Password asli tidak ditampilkan.

2

Ganti password

Pilih username, isi password baru, lalu simpan. Sistem menyimpan password sebagai hash di backend.

3

Atur permission role

Ubah Y/N untuk menentukan menu yang boleh diakses setiap role. Owner tetap full access.

4

Pantau audit & master

Audit log membantu melihat aktivitas penting, sedangkan Master Sheet menunjukkan sheet yang tersedia dan jumlah barisnya.

Catatan penting

Jangan ubah permission role saat operasional ramai kecuali sudah yakin. Setelah perubahan, minta user logout dan login ulang agar izin terbaru terbaca.

Panduan bisa dibuka lagi dari tombol Panduan Admin di sidebar.

Data User

Edit User Sistem

Ubah username, nama, level, outlet, dan password. Level dan outlet dipilih dari dropdown agar tidak salah ketik.

Password lama tidak ditampilkan demi keamanan. Untuk edit username/nama/level/outlet, backend perlu action adminUpdateUser. Jika belum terpasang, pasang Code.gs revisi yang saya sertakan.

Klik Simpan untuk memperbarui data user.
